# openpact (placeholder)

This package reserves the unscoped `openpact` name on npm. The real
OpenPact CLI is published as [`@openpact/cli`](https://www.npmjs.com/package/@openpact/cli)
once v0.1.0 ships.

For now:

```bash
npm install -g @openpact/cli
```

> **OpenPact** is a P2P daemon giving software agents a shared, append-only
> memory. Built on Hypercore + Autobase + Hyperswarm.
>
> - Source: [github.com/openpact-dev/openpact](https://github.com/openpact-dev/openpact)
> - Website: [openpact.dev](https://openpact.dev)
> - Design: [`OPENPACT_DESIGN.md`](https://github.com/openpact-dev/openpact/blob/main/docs/OPENPACT_DESIGN.md)
